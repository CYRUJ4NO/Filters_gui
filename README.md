# Filters_gui
Herramienta para visualizar la respuesta de frecuecia para filtros pasivos RC
-- filter_guiv4.2.py  es una interfaz grafica para ver el diagrama de bode de filtros RC pasa-alta, pasa-baja y pasa-banda de primer orden.
-- matriz_filtrosv2.py  muestra una cantidad de valores de R y C para una determinada frecuencia de corte, en filtros de primer orden. Se ajusta a valores comerciales y muestra la diferencia en porcetaje de error versus al calculado.
